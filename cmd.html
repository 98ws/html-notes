2. 命令行¶

ls

列出当前目录的内容

ls --help

获得帮助

ls -a或la

列出当前目录的所有内容

Ls -l 或 ll

以长格式显 示文件和目录信息，包括权限、所有者、大小、创建时间等。

cd <targetdir>

切换目录

cd <abspath>

切换目录（输入绝对路径）

cd

切换至home目录

cd ..

切换至上一层目录

cd -

切换至上一个目录

make

命令

make

查看支持的生成格式等

make html

构建HTML网页

make pdf

构建PDF

make clean

删除 build 目录中的内容

mkdir <dirname>

创建目录

mkdir -p test1/test2/test3/test4

递归创建多层目录

rm <file>

删除文件、目录

rm -r <dirname>

递归删除目录（较危险）

rm -i

删除前确认

rm -f

强制删除，不会出现警告

rmdir

删除空目录

rm --help

获得帮助

cp <sourcefile> <targetfile>

复制文件

cp <sourcefile> .

复制源文件至当前目录，命名不变

cp -r <sourcedir> <targetdir>

复制目录、文件夹（-r表递归）

cp-pr ~他人/文件夹 自己文件夹

例子：cp-pr ~xuqw/char char

mv <targetfile> <targetdir>

移动目标文件

mv <dirname> <dirname>

目录更名字

pwd

打印当前目录的绝对路径

man <command>

显示命令的手册页面 （man cp查看如何使用复制）

ln f1 f2

硬链接（f2是f1的硬连接）

ln -s f1 f3

软连接（f3是f1的快捷方式）

svn co <target>

取出目标文件（夹）、图片

svn ci <targetfile>

提交目标文件（夹）、图片

svn co http://192.168.1.100/项目名/

取出文件

svn status

查看状态 M修改

svn add <target>

新增目标文件（夹）、图片 ,属性从?变为A

svn proplist <targetfile> - svn pl

显示文档属性

svn pd <property> <targetfile>

删除文档属性 propdel

svn ps svn:eol-style LF <targetfile>

设置文档属性 propset

svn del <target>

删除文档

svn update

更新至库里最新版本

svn ci后写日志、 esc、:wq

写完日志后上传文档

svn log

查log记录

svn del <targetfile>

删除本地文件

svn ci <已删除文件名>

从库里删除文件

svn-commit.tmp

提交失败的log

!$

上一个命令行的最后一个文件

svn status|grep -v ^\?

不显示？status的文件

sdf

对比库和本地文件差别

-r 上箭头

显示添加失败的过往log

lsb-release -a

查看a120\a178等系统的信息

修改tex文件后生成pdf

输入命令行 xelatex <文件名.tex> -s -o <文件名.pdf> -s代表standalone;-o代表output

输入一小半+tab键

自动补充

gvim <文件名>

把文件用vim打开

gvim .zshrc

修改alias

vim

在putty里面编辑

:syn off

关闭语法高亮

:syn on

打开语法高亮

gvim ~/.zshrc

打开alias设置文件

gvim ~/.vimrc

打开快捷键、配置文件

fc-list :lang=zh |more

显示服务器安装的字体

pip list |grep svg

显示带有svg的文件 （如sphinxcontrib-svg2pdfconverter)

svn st | grep "^\!" | awk '{print $2}' | xargs svn delete
